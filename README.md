```
  # normal

```

# PUFSACO C++

## Ejecution
### Mac

```
  # openmp
  clang++ -Xpreprocessor -fopenmp -lomp -std=c++11 PUFSACO.cpp && time ./a.out
```

### Linux

```
  # openmp
  g++ -fopenmp PUFSACO.cpp && time ./a.out
```
